WTexScan Help

WTexScan v1.1.1 by Matt Tropiano (using DoomStruct v2.14.1)
Usage: wtexscan [--help | -h | --version] [files] [switches]

    --help              Prints help and exits.
    -h

    --version           Prints version, and exits.

[files]:
    <filename>          The files to inspect (WAD/PK3/PKE, accepts wildcards).

[switches]:
    --quiet             Output no messages.
    -q

    --textures          Output textures.
    -t

    --flats             Output flats.
    -f

    --no-skies          Skip adding associated skies per map.

    --map [mapname]     Map to scan. If not specified, all maps will be scanned.
    -m
